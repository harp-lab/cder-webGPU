Description

This application demonstrates a force-directed graph layout and rendering algorithm implemented with WebGPU. Unconnected nodes repel each other while connected nodes are pulled together, creating a physically optimal visualization.

Controls:

  • Number of Nodes: Sets the number of nodes for the next random graph generation
  • Cooling Factor: Controls how quickly the layout stabilizes (higher values = slower cooling)
  • Generate Random Graph: Creates a new random graph with the specified number of nodes
  • Run Force Directed Layout: Starts the force-directed layout algorithm
